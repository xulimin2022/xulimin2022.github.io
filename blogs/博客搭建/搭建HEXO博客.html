    搭建HEXO博客

    XuLiMin 2022/4/5 HEXO

    # 一、安装nodejs

    安装包下载地址：https://nodejs.org/en/download/

    查看node版本：node -v

    查看npm版本：npm -v

    # 二、安装Git

    安装包下载地址：https://git-scm.com/download/win

    设置用户名、邮箱 git config --global user.name "username"

    git config --global user.email "email"

    生成ssh密钥

    ssh-keygen -t rsa -C 'email'

    查看ssh密钥

    cat ~/.ssh/id_rsa.pub

    # 三、安装cnpm

    全局安装

    命令：npm install -g cnpm --registry=https://registry.npm.taobao.org

    查看cnpm版本：cnpm -v

    # 四、安装HEXO

    全局安装

    命令：cnpm install -g hexo-cli

    查看hexo版本：hexo -v

    # 五、初始化博客并启动

    新建博客资源路径：D:\xxx\blog

    在博客资源路径下执行初始化：hexo init

    在博客资源路径下执行启动：hexo s

    # 六、访问博客

    根据hexo博客启动后，输入的地址访问：http://localhost:4000

    # 七、新建博客

    在博客资源路径下执行：hexo n "第一篇博客"

    执行完毕后，在博客资源路径下：D:\xxx\blog\source_posts，会出现**”第一篇博客.md“**文件。

    清理：hexo clean

    生成：hexo g

    # 八、部署至GitHub

    首先登陆个人github账号，将生成的ssh密钥进行设置。

    新建仓库，仓库名称固定格式：xxx.github.io

    在博客资源路径下安装git部署插件：cnpm install --save hexo-deployer-git

    在博客资源路径下修改配置文件：_config.yml

    在最底部找到deploy节点进行修改

    操作 属性 属性值 备注
    修改 type git
    新增 repo git@github.com:xxx/xxx.github.io.git GitHub仓库地址
    新增 branch master

    清理一下博客：hexo clean

    重新生成：hexo g

    推送至远端执行：hexo d

    # 九、更换主题

    在博客资源路径下，执行克隆地址：git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

    在博客资源主题路径下D:\xxx\blog\themes\就可以看到yilia主题

    修改配置文件_config.yml

    操作 属性 属性值 备注
    修改 theme yilia 属性值即是主题名称

    清理一下博客：hexo clean

    重新生成：hexo g

    推送至远端执行：hexo d
