搭建HEXO博客

搭建HEXO博客步骤

一、安装nodejs

安装包下载地址：https://nodejs.org/en/download/

查看node版本：node -v

查看npm版本：npm -v

二、安装cnpm

全局安装

命令：npm install -g cnpm –registry=https://registry.npm.taobao.org

查看cnpm版本：cnpm -v

三、安装HEXO

全局安装

命令：cnpm install -g hexo-cli

查看hexo版本：hexo -v

四、初始化博客并启动

新建博客资源路径：D:\xxx\blog

在博客资源路径下执行初始化：hexo init

在博客资源路径下执行启动：hexo s

五、访问博客

根据hexo博客启动后，输入的地址访问：http://localhost:4000

六、新建博客

在博客资源路径下执行：hexo n “第一篇博客”

执行完毕后，在博客资源路径下：D:\xxx\blog\source_posts，会出现”第一篇博客.md“文件。

清理：hexo clean

生成：hexo g

七、部署至GitHub

首先登陆个人github账号，新建仓库。

仓库名称固定格式：xxx.github.io

在博客资源路径下安装git部署插件：cnpm install –save hexo-deployer-git

在博客资源路径下修改配置文件：_config.yml

在最底部找到deploy节点进行修改

操作 属性 属性值 备注
修改 type git
新增 repo https://github.com/xxx/xxx.github.io.git GitHub仓库地址
新增 branch master

部署至远端执行：hexo d

输入github账号名称、密码
